# JioSaavnAPI

## Usage

A simple usage example:

```dart
import 'package:jiosaavn_api/jiosaavn_api.dart';

main() {
  var saavn = JioSaavn();

  saavn.search('Jee le zara');
  saavn.getSongs(['iU7UPQsU', 'f6_mfbG0']);
}
```
